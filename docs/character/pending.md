# 要整理のメモ

# キャラ対策
## ゲッター1

### ゲッタービーム 214A/B/C
- 1F投げ無敵、暴れ潰し択、Cビームはヒット/ガード後空中ダッシュが可能
- Aビームはガード後+8有利
  -  青龍なら近ければ2C
- Bビームはガード後+9有利
  - バリアすると+14有利
    - 青龍なら3B
- Cビームをガード後+10有利だが、空中バックダッシュで誤魔化せる
  - バリアすると空中ダッシュが出せない、バリアすると+13有利
    - 青龍ならバリア後3B


## ゲッター2
5Aが+ガード2 5B/5Cがガード五分

5Bの対空が強い(発生後頂点付近に食らい判定がなくなる)

カグヤのJCくらい強いジャンプ攻撃の判定があれば5Bの外から攻撃が当てられる(青龍の飛びはほぼ5Bで落とされる)

ダッシュを見てから止めるはほぼ無理

236Bはガード後+3有利、5A5A5A>236Bのような連携は連続ガードでゲッター2側が入れ込める連携

5A1段目が+2,2段目が-1,3段目が-3

### 5A>5A5A5A>236A
- ゲッタードリルは強襲で割れる
  - 青龍の場合5Cや2Cでも割れる 


### 5A5A5A>214A
- 下から出てくるドリルはガードで+5フレ有利は強襲で割れる
- ドリル後、後ろゲッタービジョンで誤魔化してくるような場合は突進技でリスクを追わせられる
  -  青龍の4溜め6B等

### 6B
- 1F空中判定になる技、カウンターでコンボになる
  -  主に投げ潰しの択



## ゲッター3

### 5A>2A
- 2Aの後5Dなら連続ガードでガード後4F有利
  - 2Aの後無敵技が仕込み得
- 2Aの後ゲッターミサイル(A/B/C)なら突進技等で割る、EXミサイルだと状況による
  - 青龍なら3B
  - カグヤなら214B
- 2Aの後プレス
  - Aプレスは連続ガード、ガード後13F有利なのでチェンジで隙を消してくる
  - Bプレスはガード後17F有利、ガードされてたらチェンジで隙を消してくる
  - Cプレスはガード後13F有利、めくり攻撃でチェンジで隙を消してくる
    - A/Bプレスはチェンジ後こちら側が有利
    - Cプレスのチェンジ後は相手が有利っぽい

### 5B
- 移動しながら伸ばしてくる
- ガード後7F有利
- 伸びてくる手にFC対応の技を置いておくとリターン出るので、試す価値はある

### 2C>ゲッターミサイル
- ミサイル見てから突進系の技
  - 青龍ならミサイル見てから3B
- EXミサイルだと発生が早く後ろに下がり始めるのが早いので当たらない
